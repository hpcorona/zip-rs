Info
----

A zip library for rust wich supports reading and writing of simple ZIP files.

Supported compression formats:

* stored (i.e. none)
* deflate
* bzip2 (removed)

Currently unsupported zip extensions:

* ZIP64
* Encryption
* Multi-disk
